### 프로젝트 목적
- 유치원 교사의 편의를 위한 부모 대상의 스토리 가이드 제공을 목표로 하는 웹

### 빠른 시작 
1. Node.js 및 npm 설치
2. 의존성 설치: `npm install`
3. 개발 서버 시작: `npm run dev`  (기본 포트: `8080`)

### 빌드 및 미리보기
- 빌드: `npm run build`
- 미리보기: `npm run preview`
